I asked for plain language over jargon more than once — when an assistant reaches for
technical terms I don't use myself, I have to stop and ask for a translation.

**Why:** Jargon slows me down mid-task; I'd rather read one clear sentence than look up
a term.
**How to apply:** Default to everyday words. If a technical term is unavoidable, define
it in the same sentence.
